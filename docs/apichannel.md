Channel is a platform where the users send and receive messages. Members can be added and removed from an existing channel. You can create and define the type of channels depending on the purpose of the channel like channels for a team, organizational purposes, collaboration, external organizations. Based on your requirements you can modify the channel and add members and assign particular permissions to them.

There are three types of channels:

- Public
- Private
- Direct

### Get channels By label ###

You can get channels by label, using the following Url endpoint.



Add the following endpoint to the base Url.

/channels?filter=type:eq:private


|Filter|Description|HTTP request|
|---|---|---|

|in|in a specific type of channel||
|begins|get channels with subject beggining with a certain letter||
|ends|channel subject ending with a certain letter||
|contains|contains|channel containing symbol|
